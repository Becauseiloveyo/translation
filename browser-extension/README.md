# LiteDict Browser Extension

This is a Chrome/Edge Manifest V3 extension for Windows.

## Install Locally

1. Open Chrome or Edge.
2. Go to `chrome://extensions` or `edge://extensions`.
3. Enable Developer mode.
4. Choose Load unpacked.
5. Select this folder: `browser-extension`.

## Features

- Reads selected text from the active page.
- Translates with a mock provider by default.
- Supports an OpenAI-compatible chat completions provider configured in the extension options page.
- Stores settings and history in browser local storage.

No API keys, dictionary data, or third-party brand assets are bundled.
