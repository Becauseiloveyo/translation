# LiteDict mobile polish build

This release publishes the latest LiteDict mobile build after the recent dictionary, vocabulary, backup, and visual polish work.

## Highlights

- Improves mobile dictionary suggestions with strict prefix matching.
- Enlarges the dictionary search touch target for easier mobile use.
- Adds dictionary import preview, conflict handling, template downloads, and priority ordering.
- Connects vocabulary cards back to dictionary lookup and saves richer vocabulary metadata.
- Adds backup restore preview and release preflight checklist support.
- Fixes dark mode visual hierarchy and settings dropdown flow.
- Keeps the existing launcher icon configuration unchanged.

## Manual checks after install

- Verify the launcher icon remains unchanged.
- Check dictionary lookup for `auto`, `automatic`, and `running`.
- Check translation language dropdowns on mobile.
- Check settings dropdowns push cards down instead of overlapping.
- Check dark mode on dictionary, translation, vocabulary, and settings pages.
